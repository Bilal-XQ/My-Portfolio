# Portfolio Personnel

Un portfolio moderne et interactif développé avec HTML5, TailwindCSS et JavaScript, présentant mes compétences en développement web, mes projets et mon parcours académique avec une interface élégante et des animations fluides.

## 🚀 Fonctionnalités

- Design responsive et moderne
- Mode sombre/clair
- Animations fluides et interactives
- Sections détaillées (À propos, Compétences, Projets, etc.)
- Formulaire de contact fonctionnel
- Optimisation des performances (lazy loading, etc.)

## 🛠️ Technologies Utilisées

- HTML5
- TailwindCSS
- JavaScript (Vanilla)
- GSAP pour les animations
- AOS (Animate On Scroll)

## 📦 Installation

1. Clonez le repository
```bash
git clone https://github.com/votre-username/portfolio.git
```

2. Installez les dépendances
```bash
npm install
```

3. Lancez le serveur de développement
```bash
npm run dev
```

## 🔧 Configuration

1. Modifiez les informations personnelles dans `index.html`
2. Ajustez les styles dans `css/styles.css`
3. Personnalisez les animations dans `js/animations.js`

## 📱 Compatibilité

- Chrome (dernière version)
- Firefox (dernière version)
