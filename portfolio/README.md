# Portfolio Personnel

Ce projet est un portfolio en ligne pour présenter mes compétences, projets et expériences professionnelles.

## Structure du Projet

```
portfolio/
│
├── index.html                 # Application monopage
├── favicon.ico                # Favicon du site
│
├── css/
│   ├── style.css              # Styles personnalisés
│   └── tailwind.config.js     # Configuration Tailwind
│
├── js/
│   ├── main.js                # Logique JavaScript principale
│   ├── animations.js          # Effets d'animation
│   ├── projects.js            # Logique de présentation des projets
│   └── skills.js              # Visualisation des compétences 
│
├── assets/
│   ├── images/                # Images du portfolio
│   ├── resume/                # CV téléchargeable
│   └── fonts/                 # Polices personnalisées
│
└── README.md                  # Documentation du projet
```

## Technologies Utilisées

- HTML5
- CSS3 avec Tailwind CSS
- JavaScript

## Comment Utiliser

1. Clonez ce dépôt
2. Ouvrez `index.html` dans votre navigateur
